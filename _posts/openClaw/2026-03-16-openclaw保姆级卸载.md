# OpenClaw 官方卸载教程（保姆级）

> 本教程将指导你彻底卸载 OpenClaw，包括网关服务、本地数据库、配置文件等所有数据。

---

## 📋 卸载步骤

### 步骤 1：打开终端（Terminal）

根据你的操作系统，选择以下方式之一：

#### Windows 用户

**方式一：使用运行对话框**
1. 按键盘上的 `Win + R` 键
2. 在弹出的框中输入 `cmd`
3. 按回车键

**方式二：使用开始菜单**
1. 点击开始菜单
2. 搜索"命令提示符"或"PowerShell"
3. 点击打开

#### Mac 用户

1. 按键盘上的 `Command + Space`（空格键）
2. 在搜索框输入 `Terminal` 或 `终端`
3. 按回车键

---

### 步骤 2：执行卸载命令

在终端中输入以下命令并回车：

```bash
openclaw uninstall --all --yes
```

#### 参数说明

| 参数 | 说明 |
|------|------|
| `uninstall` | 告诉程序我要卸载 |
| `--all` | 彻底删除，包括网关服务、本地数据库、配置文件等所有数据 |
| `--yes` | 全程自动确认，不需要你手动按 Y 确认 |

> ⏱️ **等待完成**：命令执行可能需要几分钟，请耐心等待直到终端显示完成提示。

---

### 步骤 3：删除命令行工具

上面的指令运行完后，你的电脑里就只剩下 OpenClaw 的外壳（CLI 工具）了。如果你想把它也彻底送走，再执行一行命令：

```bash
npm uninstall -g openclaw
```

---

### ✅ 卸载完成

至此，你的电脑就彻底干净了！

---

## ⚠️ 注意事项

### 1. 重启电脑

执行完上述操作后，**建议重启一次电脑**，确保所有后台进程彻底关闭。

### 2. 废弃 API Key（重要！）

如果你曾经在 OpenClaw 里绑定过 API Key（如 OpenAI、Claude 的密钥），**强烈建议**去对应的官网废弃旧 Key，生成新的 Key，以防万一。

**操作步骤：**

| 服务商 | 操作链接 |
|--------|----------|
| OpenAI | https://platform.openai.com/api-keys |
| Anthropic (Claude) | https://console.anthropic.com/settings/keys |
| 其他服务商 | 登录对应官网的 API 设置页面 |

### 3. 数据备份（卸载前）

如果你以后可能重新使用 OpenClaw，建议在卸载前备份重要数据：

```bash
# 备份工作区
cp -r ~/.openclaw/workspace ~/openclaw-workspace-backup

# 备份配置文件
cp ~/.openclaw/config.json ~/openclaw-config-backup.json
```

---

## 🔄 重新安装

如果你之后想重新使用 OpenClaw，可以执行：

```bash
# 安装 OpenClaw CLI
npm install -g openclaw

# 初始化
openclaw init
```

## ❓ 常见问题

### Q: 卸载后还能恢复数据吗？
A: 使用 `--all` 参数后，所有数据会被永久删除，无法恢复。请务必在卸载前备份重要数据。

### Q: 卸载命令执行失败怎么办？
A: 尝试以管理员身份运行终端：
- **Windows**：右键点击"命令提示符"或"PowerShell"，选择"以管理员身份运行"
- **Mac**：在命令前加 `sudo`，如 `sudo openclaw uninstall --all --yes`

### Q: 如何确认是否卸载干净？
A: 执行以下命令检查：
```bash
# 检查 CLI 是否存在
openclaw --version

# 检查网关服务是否运行
openclaw gateway status
```
如果提示"command not found"或类似错误，说明已卸载干净。
